Szélerőművek – Web-programozás 1 beadandó
Projekt leírása

Ez a projekt a Web-programozás 1 tantárgy beadandó feladataként készült.
A webalkalmazás témája a magyarországi szélerőművek adatainak kezelése és bemutatása.

Az alkalmazás PHP alapú webes rendszer, amely adatbázis-kezelést, felhasználói azonosítást és CRUD műveleteket valósít meg.

Projekt struktúra
includes/ -> adatbázis kapcsolat, konfiguráció
logicals/ -> üzleti logika
templates/ -> oldalak megjelenítése
styles/ -> CSS fájlok
images/ -> képek és galéria
videos/ -> videófájlok
